<h1>🌐 My Static Website</h1>

<p>A simple, responsive, and lightweight static website built using 
<strong>HTML</strong>, <strong>CSS</strong>, and <strong>JavaScript</strong>.  
This project is designed to be fast, minimal, and easily customizable.</p>

<hr>

<h2>✨ Features</h2>
<ul>
  <li>📱 <strong>Responsive Design</strong> – works across desktop, tablet, and mobile.</li>
  <li>🎨 <strong>Modern UI</strong> – clean and minimal styling with reusable components.</li>
  <li>⚡ <strong>Lightweight</strong> – no frameworks, just vanilla HTML, CSS, and JS.</li>
  <li>🔍 <strong>SEO-Friendly</strong> – proper meta tags and structure for search engines.</li>
  <li>🚀 <strong>Fast Performance</strong> – optimized assets for quick load times.</li>
</ul>
